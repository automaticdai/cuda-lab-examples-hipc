# cuda-lab-examples

Examples for the UoY HIPC CUDA course (lecture 7).

## Usage

In a terminal:

`$ make`

Then run the program in `bin/`.

Make sure you have nvidia-driver and cuda-toolkit installed.

To test the environment, run:

`$ nvidia-smi`

and 

`$ nvcc --version`

## Run Jobs on Viking

`$ sbatch cuda_viking.job`
